Movie webpage project

Context

This project is challenging students to make a movie webpage using their current basic knowledge of CSS and HTML coding to design a web-page to promote their upcoming movie,the students work in pair to create a movie-webpage using what they had learnt over the past 2-3 months

Challenges,Takeaways,Next steps

I made a movie-webpage that's action related along with my partner,a boy named williams, discovered the harsh truth that he isn't living with his biological father, and seeks to find the truth and learn more about his real father We faced some trouble as At first I didn't know to redirect people to the sequel as the main page and sequel shares the same link but I did a little more digging and figured out you can use the title of the files names as links to redirect people, Overvall I improved by CSS and HTML skills and got a little better at solving issues that I faced, If I were given a little more time, I would work on he plot of the story, a little more work on the CSS.
